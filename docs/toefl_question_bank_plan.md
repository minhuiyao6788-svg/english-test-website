# 托福TOEFL测试题库创建计划

## 任务概述
创建一个完整的托福TOEFL测试题库，包含基础版50题和完整版100题，总计150道题目。

## 任务要求
- 基础版50题：词汇题(15题)、语法题(15题)、阅读理解题(20题)
- 完整版100题：词汇题(30题)、语法题(30题)、阅读理解题(40题)
- 所有题目均为单选题，每题4个选项
- 包含标准答案和详细解析
- 题目难度符合托福标准，贴近真实考试
- 保存为JSON格式文件：data/toefl_questions.json
- 包含题目分类、难度等级、知识点标签等元数据
